** STREAM API  **

Veri işlemek için kullanılır. List, set , map gibi koleksiyonlarda.
 Kolay okunabilirlik sağlar.

 streamexample package ile Integer bir liste üzerinden genel stream operasyonları kullanıldı.
   * map , 
   * reduce ile tek değere indirme ,
   * anymatch ile boolean değerinde listede var mı yok mu kontrolü yapıldı

  streambank package ile Customer, Account, Transaction class ları oluşturularak bankacılık örnekleri yapıldı
  Yukarıdaki operasyonlara ek olarak 
  * filter 
  * sum 
  * groupingBy 
  * groupingBy ile birlikte kullanılan operasyonlara örnekler
     ** mapping() 
     ** flatMapping
     ** filtering 
     ** teeing 
  * sorted()
  * skip 
  * limit 
  * anyMatch, noneMatch, allMatch
